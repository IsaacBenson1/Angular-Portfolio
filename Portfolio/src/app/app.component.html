JavaEngine

"JavaEngine" is the name of a project that I started in June of 2021. It is a 3D engine written entirely from scratch in java. It uses properties of linear algebra and projective geometry in order to achieve a form of rendering called rasterization. This engine is capable of loading obj files from blender, and has a movable camera. It uses no external libraries other than JFrame, which is just the window that the program runs in.
